# The docs folder

</div>

Files in this folder are automatically uploaded to https://matrix4348.github.io/Dragons-of-the-Void---Raid-Loot-Tiers/ in order to be downloaded by the user script when needed (on load and every hour, for those who play long enough withut refreshing the page). This ensures players always have the latest data available instead of forcing them to wait for a script update.

It contains the following files:
* [raid-list.json](documentation/raid-list-json.md)
* [raid-filters.json](documentation/raid-filters-json.md)
